Java – Métodos (W3Schools)

Este repositório contém exercícios de **Java** focados no estudo de **métodos**, feitos a partir do tutorial do W3Schools.

Aqui estão exemplos práticos para entender como métodos funcionam em Java, como passar parâmetros, retornar valores e aplicar conceitos como sobrecarga e recursão.

Conteúdos praticados

Os exercícios abordam:

- Criação e chamada de métodos
- Parâmetros de métodos
- Passagem de valores como parâmetros
- Retorno de valores
- Sobrecarga de métodos (overloading)
- Recursão

Organização dos arquivos

Cada arquivo `.java` representa um conceito específico:

- `JavaMethods.java` — introdução e uso básico de métodos  
- `MethodParameters.java` — uso de parâmetros  
- `MethodParametersValues.java` — passagem de valores para métodos  
- `Overloading.java` — sobrecarga de métodos  
- `Recursion.java` — métodos recursivos  

Objetivo do repositório

Este repositório serve como um registro prático do meu aprendizado em Java, com foco em métodos e na lógica por trás de cada exemplo.

Também funciona como material de revisão e consulta para estudos futuros.

Observação

Este repositório tem fins educacionais e acompanha meu progresso durante o estudo de Java pelo W3Schools.
